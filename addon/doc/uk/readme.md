# Вдосконалення промовляння введених слів - доповнення NVDA

NVDA читає все слово, коли воно змінюється, а не лише додану частину, що є типовою поведінкою.

Цей код було отримано з надбудови NVDAExtensionGlobalPlugin Paulber19.
https://github.com/paulber19/NVDAExtensionGlobalPlugin
Відповідно до умов Загальної громадської ліцензії GNU (GNU GPL).
Усі заслуги належать її початковим авторам — Paulber19 та Leonardder.

Функцію, яка виконувала цей код, було вилучено з оригінального додатка у версії 13.3, однак деякі користувачі сумували за нею.
Цей форк відновлює функцію, адаптувавши її для самостійної роботи у вигляді невеликого додатка для тих користувачів.

### Попередження:  

* Якщо запущено старішу версію **NVDAExtensionGlobalPlugin** і в ній увімкнено параметр **покращене озвучення слів під час введення**, обробка буде передана цьому додатку. Коли параметр вимкнено або додаток **NVDAExtensionGlobalPlugin** видалено чи оновлено, керування озвученням слів під час введення (**speakTypingWords**) перейде до цього додатка.

* Повідомлялося про помилки під час використання цієї функції в **MuseScore**, які неможливо було виправити. Тому вона залишатиметься вимкненою в цьому застосунку, доки проблеми не буде усунуто.

* Деякі користувачі повідомляли про проблеми з використанням функції **покращеного озвучення слів** із **Брайлівськими клавіатурами**. Я не можу гарантувати, що вона працюватиме коректно з усіма Брайлівськими пристроями.
